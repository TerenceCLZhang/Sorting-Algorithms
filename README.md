- Bubble Sort
- Optimized Bubble Sort
- Odd Even Sort
- Comb Sort
- Cocktail Sort
- Gnome Sort

- Binary Insertion Sort
- Shell Sort

    - Shell (1959)
    - Hibbard (1961)

    - Knuth (1973)
    - Sedgewick (1982)
    - Sedgewick (1986)
    - Tokuda (1992)
    - Ciura (2001)
    - Lee (2021)
    - Skean, Ehrenborg, and Jaromczyk (2023)
- Selection Sort
- Heap Sort
